# Moving Things with JavaScript by Acting on Events

# ExD – Akvarie (Forbedret Eksamensversion)

Dette projekt er en forbedret version af mit oprindelige ExD-akvarie udviklet på Multimediedesigneruddannelsens 1. semester. Målet har været at skabe en mere legende, intuitiv og børnevenlig oplevelse for målgruppen 4–6 år gennem forbedret UX/UI, animationer og interaktivitet.

## Formål
- Skabe et engagerende, visuelt og letforståeligt univers for børn.
- Forbedre både brugeroplevelse, funktionalitet og kodekvalitet.
- Implementere UX-principper til børn: simplicity, feedback, playfulness og consistency.

## Hvad er forbedret?
- **UX/UI:** større klikflader, bedre farvekontraster og tydeligere affordance.
- **Animationer:** glattere fiskebevægelser og sway-effekt på koraller.
- **Lydfeedback:** bobler, piratskib, skattekiste og spillyde tilføjet.
- **Fakta-boble:** ny pop-up med glidende animation og børnevenlig tekst.
- **Spillet:** stabil kollisionsdetektion, bedre flow og forbedret respons.
- **Kode:** opryddet mappestruktur, bedre navngivning og optimeret CSS/JS.

## Teknologier
- **HTML5** – struktur og semantik  
- **CSS3** – animationer, layout og visuel identitet  
- **JavaScript** – interaktion, logik, lyd og spilfunktioner  

## Målgruppe
Projektet er designet til børn i alderen 4–6 år, baseret på udviklingspsykologiske principper (Piaget) og UX-design for børn.
